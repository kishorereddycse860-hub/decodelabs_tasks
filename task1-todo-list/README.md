# Task 1: To-Do List Application

A simple command-line To-Do List app built in Python.

## Features
- Add tasks to your list
- View all tasks with numbering
- Exit the program anytime

## How to Run
python todo_list.py

## Menu Options
1. Add a task – Enter a task description to add it to your list
2. View all tasks – Displays all tasks currently in your list
3. Exit – Closes the program

## Example
====TO-DO-LIST====
1.Add a task:
2.View all tasks:
3.Exit
choose an option(1/2/3):1
Enter the task you want to add:Finish DecodeLabs internship
'Finish DecodeLabs internship' has been added.

## Tech Used
- Python 3 (built-in input(), list operations, loops)
